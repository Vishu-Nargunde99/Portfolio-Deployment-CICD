What is Amazon EC2?

Amazon EC2 (Elastic Compute Cloud) provides secure, resizable compute capacity in the cloud. It is designed to make web-scale cloud computing easier for developers by allowing them to launch virtual servers with full control over compute resources.

Key benefits include flexible instance types, automatic scaling, integration with storage and networking services, and pay-as-you-go pricing. EC2 is commonly used to host applications, web servers, development environments, and more.

Amazon S3 (Simple Storage Service) is a highly scalable object storage service designed for storing any amount of data from anywhere. It's commonly used for backup, data archiving, hosting static websites, and as storage for other AWS services.

RDS

Managed relational database service for easy deployment and scalability.

Amazon RDS (Relational Database Service)

Amazon RDS is a managed database service that makes it easy to set up, operate, and scale a relational database in the cloud. It supports several engines like MySQL, PostgreSQL, MariaDB, Oracle, and SQL Server.

üîç Key Features:
  • Automated backups, patching, and failover
  • High availability with Multi-AZ deployments
  • Easy scaling for storage and compute
  • Secure access using VPC, IAM, and encryption
üöÄ Projects Using RDS:

  • Purpose: Shows how to decouple each layer for scalability and security.

What is Amazon VPC?

Amazon Virtual Private Cloud (VPC) allows you to launch AWS resources in a logically isolated virtual network that you define. It gives you complete control over your networking environment, including IP address ranges, subnets, route tables, and gateways.
